    • What's an Ukagaka/ghost and how do I use it?

    An Ukagaka is a character that sits on your desktop and keeps you company. You'll need a program to run it:

    Windows: SSP -choose 'complete install package'

    Mac: Ukagaka on Macs, NiseRingo, PlayonMac+SSP, WineBottler

    Linux: NINIX-AYA

    Once you've installed SSP, run it and the default character will appear- Emily the catgirl! Wait for her to finish her introduction.

    Now you'll want to install Zilch's files by dragging and dropping them on her. Zilch uses z_undertale balloon by Zarla- you should install it first if you don't have it already. Then, install Zilch's .nar file. Once Emily's done, right click her to change ghost.

    Troubleshooting

  • An error window in Japanese appears when starting SSP:

    Drop the .nar file onto that window. If that doesn't work, click the first button and point it towards your ghost directory(something like \SSP\ghost). The second button is for the balloons- point it to \SSP\balloon.

  • The right click menu is in Japanese:

    The language option is the 4th one up from the bottom. Select English and restart SSP.

  • The catgirl won't install the ghost for you:

    First, unzip the z_undertale balloon to get the folder 'z_undertale'. Paste it into the \SSP\balloon folder next to emily's folder.

    Then, unzip the .nar file to get the folder 'ds_zilch' containing the subfolders 'ghost' and 'shell'. Paste the 'ds_zilch' folder into the \SSP\ghost folder next to emily's folder.

    Restart SSP and check the right click menu again.
